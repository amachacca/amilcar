ASSETS FOLDER
=============

This folder should contain the following images:

1. profile.jpg - Your profile photo (recommended: 200x200px, square)
2. project-1.jpg - Screenshot of first project (recommended: 400x260px)
3. project-2.jpg - Screenshot of second project (recommended: 400x260px)
4. project-3.jpg - Screenshot of third project (recommended: 400x260px)
5. resume.pdf - Your CV/Resume file for download

Image Recommendations:
- Use optimized images (compress with TinyPNG or similar)
- Keep file sizes under 200KB for fast loading
- Use JPG for photos, PNG for graphics with transparency

To use:
1. Add your images to this folder
2. Make sure the filenames match exactly
3. Deploy to GitHub Pages
